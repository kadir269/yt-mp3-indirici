# 🎵 YouTube MP3 İndirici - Frontend-Only

Modern, güvenli ve frontend-only YouTube'dan MP3 indirme uygulaması. Backend yok, tamamen Vercel'de çalışır.

## ✨ Özellikler

- 🔒 **Güvenli** - Çerezsiz çalışır, backend yok
- 🚀 **Hızlı** - Cobalt.tools API ile yüksek hızda indirme
- 🎨 **Modern Arayüz** - Şık ve kullanıcı dostu tasarım
- 🖼️ **Video Önizleme** - İndirmeden önce video bilgilerini görüntüleme
- 🍪 **Çerez Opsiyonel** - Sadece bazı videolarda gerekebilir
- 📱 **Responsive** - Mobil uyumlu tasarım

## 🚀 Vercel'e Deploy

### 1. Vercel'e Git
- [vercel.com](https://vercel.com) adresine git
- GitHub hesabınla giriş yap

### 2. Yeni Proje Oluştur
- "Add New Project" butonuna tıkla
- GitHub reposunu seç (veya manuel yükle)

### 3. Deploy Et
- "Deploy" butonuna tıkla
- Otomatik olarak deploy olacak
- URL'yi al

### 4. Test Et
- Siteyi aç
- YouTube URL'si dene

## 📖 Nasıl Kullanılır?

### Video İndirme

1. **URL Girin:** YouTube video URL'sini yapıştırın
2. **Ara:** "Ara" butonuna tıklayın
3. **İndir:** "MP3 İndir" butonuna tıklayın
4. **Tamam:** Dosya otomatik indirilir

## 🔒 Güvenlik Notları

- **Çerezler:** Sadece localStorage'da saklanır
- **Backend Yok:** Hiçbir sunucuya veri gönderilmez
- **Risk Yok:** Tamamen güvenli, risk yok
- **Gizlilik:** Veriler tarayıcınızda kalır

## 🛠️ Teknolojiler

- **HTML/CSS/JS** - Frontend-only
- **Cobalt.tools API** - YouTube indirme
- **localStorage** - Çerez saklama
- **Vercel** - Hosting
