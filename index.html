🍪 Çerez Dosyası (Zorunlu)

Çerez yüklemelisiniz, çerezsiz çalışmaz:

Çerez yüklenmedi (zorunlu)

💻 Bilgisayar İçin Çerez Alma:

  1. Chrome Web Store'dan "Get cookies.txt LOCALLY" eklentisini indir
  2. YouTube'a git ve oturum aç
  3. Eklenti ikonuna tıkla
  4. "Export" → "cookies.txt" indir
  5. Yukarıdaki "📁 Çerez Yükle" butonuna tıkla
  6. cookies.txt dosyasını seç

📱 Telefon İçin Çerez Alma:

  1. Google Play Store'dan "Cookie Editor" uygulamasını indir
  2. YouTube uygulamasını aç ve oturum aç
  3. Cookie Editor uygulamasını aç
  4. YouTube çerezlerini dışa aktar
  5. Dosyayı bilgisayara gönder
  6. Yukarıdaki "📁 Çerez Yükle" butonuna tıkla
  7. Dosyayı seç
